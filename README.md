How-to
---------------------------

1. Install Python 3.x (anaconda recommended)

3. Install <>
4. Clone the repository:

   ```bash
   $ git clone <>
   ```

   or download as zip and extract.

5. In the root directory run

   ```bash
   $ python <>
   ```

1. Use <kbd>Esc</kbd> to close the program.
